# Privacy Policy — PRitty

**Last updated:** February 16, 2026

## Overview

PRitty is a browser extension that enhances the GitHub Pull Request UI with quick actions and productivity features. It operates entirely within your browser and does not collect, store, or transmit any user data.

## Data Collection

PRitty does **not** collect any data. Specifically:

- No personal information is collected
- No browsing history is tracked
- No analytics or telemetry is sent
- No cookies are set
- No data is stored locally or remotely
- No data is shared with third parties

## Permissions

PRitty requests the following browser permission:

- **activeTab** — Used solely to modify the appearance and behavior of GitHub Pull Request pages (`github.com/*/pull/*`) while you are viewing them. No data from the page is read, stored, or transmitted.

## How It Works

PRitty injects CSS and JavaScript into GitHub Pull Request pages to add UI enhancements such as action buttons, scroll navigation, and layout improvements. All processing happens locally in your browser tab. The extension does not communicate with any external server.

If this policy changes, the update will be reflected in this file with a new "Last updated" date.
